# pnp


This action produces a text file that summarizes metrics for Plug and Play events.

``` syntax
-a pnp [-range T1 T2]
```

## Options


<a href="" id="-ranget1-t2"></a>**-range***T1 T2*  
Show events between *T1* and *T2*, in microseconds.

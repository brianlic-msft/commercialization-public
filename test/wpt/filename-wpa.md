# filename


This action produces a text report that summarizes the file names in the trace.

``` syntax
-a filename [-range T1 T2]
```

## Options


<a href="" id="-ranget1-t2"></a>**-range***T1 T2*  
Shows file names recorded between *T1* and *T2*, in microseconds.
